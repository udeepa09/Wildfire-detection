# 🔥 Wildfire Detection System

##  Overview
This project is an AI-based system that detects fire and smoke from images.  
It helps in early wildfire detection and reduces environmental damage by providing fast and accurate results.

---

## Features
- Detects fire and smoke in images  
- Fast and accurate detection  
- Upload image from system or use URL  
- Displays bounding boxes on detected areas  
- Allows downloading the result image  

---

## 🛠️ Technologies Used
- Python  
- Streamlit  
- YOLO   
- OpenCV  

---

## ⚙️ How It Works
1. User uploads an image or provides a URL  
2. Image is preprocessed  
3. Model analyzes the image  
4. Fire/smoke is detected  
5. Bounding boxes are drawn on detected areas  
6. Final output is displayed
